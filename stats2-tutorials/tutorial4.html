Tutorial 4

Introduction

In this tutorial we will study two datasets. The first dataset is the penguins.csv dataset which featured on the resit exam from January 2022. This dataset is based on a dataset used in the Programming and Quantitative Skills course to learn about plotting. The data contain information from a random sample of 333 Antarctic penguins. The dataset contains 4 variables:

  • flipper: This is the penguin’s flipper length measured in mm (the penguin’s wing).
  • weight: The penguin’s weight measured in grams
  • female: A dummy variable that equals 1 if the penguin is female and 0 if it is male.
  • species: A character variable indicating the penguin’s species. It is one of "Adelie", "Chinstrap" or "Gentoo".

The second dataset is the ebill90.csv dataset from the resit exam in January 2019. It contains the monthly average household electricity bill in the USA from January 1991 to December 2000.

The variables are:

  • year: The year (as a numeric variable).
  • month: The month (as an abbreviated character variable).
  • avg_bill: The monthly average household electricity bill (in dollars).

Exercises

With the penguins.csv dataset:

  1. Estimate a regression explaining flipper length with weight, gender and their interaction. Is there evidence of any gender effect at the 5% level (that is, given a particular weight, does the flipper length differ by gender on average?). Use the critical value approach.
  2. Estimate a regression explaining flipper length with weight, gender, the species dummies and the interactions of the species dummies with gender. Use adelie penguins as the base group. Test at the 5% level if the average flipper length of female adelie penguins is larger than the average flipper length of male adelie penguins of the same weight. Use the p-value approach.

With the ebill90.csv dataset, create a trend term (a variable that indicates the row number) and regress avg_bill on the trend and all month dummies except December (use December as your base category). If you estimated this model correctly your estimated intercept should be 137.21747. Use this model to answer the following questions:

  1. In which month of the year is the average electricity bill largest? Does this make sense?
  2. Use a suitable plot to visually check for independence of the residuals.
  3. Perform a formal test for first-order autocorrelation of the residuals at the 5% level. Use the p-value approach.

Tip: Use df$month <- factor(df$month, levels = unique(df$month)) to make the order of the levels be "Jan", "Feb", …, "Nov", "Dec" instead of the month names alphabetically. Then use the relevel() function to change the base category to "Dec".

Solutions

Exercise 1

The model is: \mathbb{E}[Y_i|x_{i1},x_{i2}] = \beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \beta_3 x_{i1} x_{i2} where Y_i is flipper length, x_{i1} is weight and x_{i2} is the female dummy. For female penguins: \mathbb{E}[Y_i|x_{i1},x_{i2}=1] = \beta_0 + \beta_1 x_{i1} + \beta_2 + \beta_3 x_{i1} For male penguins: \mathbb{E}[Y_i|x_{i1},x_{i2}=0] = \beta_0 + \beta_1 x_{i1} For a fixed weight x_{i1}, the difference is: \mathbb{E}[Y_i|x_{i1},x_{i2}=1]- \mathbb{E}[Y_i|x_{i1},x_{i2}=0]= \beta_2 + \beta_3 x_{i1} If there is no gender effect whatsover, then both \beta_2=0 and \beta_3=0 and the expected flipper length given the penguin’s weight should be the same. That is to say, when \beta_2=\beta_3=0 we have \mathbb{E}[Y_i|x_{i1},x_{i2}=1]=\mathbb{E}[Y_i|x_{i1},x_{i2}=0] If there is a gender effect, then at least one of \beta_2\neq 0 or \beta_3\neq 0.

So the null and alternative hypotheses are:

\begin{split} H_0:&\beta_2=\beta_3=0 \\ H_1:&\text{ at least one of }\beta_j\neq 0 \text{ for } j=2,3 \\ \end{split} We can test this using a partial F-test because it involves testing whether a subset of our parameters are zero or not. Under H_0: F=\frac{\left(SSE_r-SSE_c\right)/\left( k-g \right)}{SSE_c/\left(n-k-1 \right)} \sim F_{k-g,n-k-1} where k=3 and g=1. We can estimate the complete and reduced models in R and use the anova() function to calculate the F test statistic. The complete model again is: \mathbb{E}[Y_i|x_{i1},x_{i2}] = \beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \beta_3 x_{i1} x_{i2} And the reduced model is: \mathbb{E}[Y_i|x_{i1}] = \beta_0 + \beta_1 x_{i1} We estimate these in R with:

df <- read.csv("penguins.csv")
m <- lm(flipper ~ weight * female, data = df)
m_r <- lm(flipper ~ weight, data = df)
anova(m_r, m)

The F statistic is 12.359. To get the critical value we can use k-g=2 numerator degrees of freedom (which can be read under Df in the anova() table) and n-k-1=329 denominator degrees of freedom (which can be read under Res.Df for model 2 in the anova() table):

Because the test statistic is larger than the critical value, we reject H_0. There is evidence for a gender effect at the 5% level.

Exercise 2

The model we want to estimate is:

\begin{split} \mathbb{E}[Y_i|x_{i1},x_{i2},x_{i3},x_{i4}] =&\beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \beta_3 x_{i3} + \\ &\beta_4 x_{i4} + \beta_5 x_{i2}x_{i3} + \beta_6 x_{i2}x_{i4} \end{split} where x_{i1} is weight, x_{i2} is the gender dummy, x_{i3} is a Chinstrap dummy and x_{i4} is a Gentoo dummy.

Because Adelie is first alphabetically, it will automatically be chosen as the base category. We can therefore estimate the model with:

m <- lm(flipper ~ weight + female * species, data = df)

For adelie penguins, the model simplifies to: \mathbb{E}[Y_i|x_{i1},x_{i2},x_{i3}=0,x_{i4}=0] =\beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} For female adelie penguins, the model is: \mathbb{E}[Y_i|x_{i1},x_{i2}=1,x_{i3}=0,x_{i4}=0] =\beta_0 + \beta_1 x_{i1} + \beta_2 For male adelie penguins, the model is: \mathbb{E}[Y_i|x_{i1},x_{i2}=0,x_{i3}=0,x_{i4}=0] =\beta_0 + \beta_1 x_{i1} Taking differences:

\begin{split} &\mathbb{E}[Y_i|x_{i1},x_{i2}=1,x_{i3}=0,x_{i4}=0] - \\&\mathbb{E}[Y_i|x_{i1},x_{i2}=0,x_{i3}=0,x_{i4}=0] =\beta_2 \end{split} So \beta_2 represents the average difference between female and male adelie penguins of the same weight.

Therefore the claim is asking if \beta_2>0. This is the alternative hypothesis. We are now ready to perform the test.

The null and alternative hypotheses are:

\begin{split} H_0:&\beta_2 \leq 0\\ H_1:&\beta_2 > 0\\ \end{split} Under H_0, T=\frac{B_2}{S_{B_2}}\sim t_{n-k-1}. The value of the test statistic can be read from the table (0.001) or extracted directly with:

t <- coef(summary(m))["female", "t value"]
t
[1] 0.001060368

The p-value for this upper-tail test can be calculated with:

1 - pt(t, m$df.residual)
[1] 0.4995773

The p-value is greater than the significance level of 1%. We therefore fail to reject the null hypothesis. There is not sufficient evidence for the claim.

Exercise 3

We first read in the data and get the variables ready to be used in the regression model before estimating it:

df <- read.csv("ebill90.csv")

# Create trend term (using the row number):
df$t <- 1:nrow(df)

# Convert month to a factor variable specifying the order of the levels to be
# "Jan", "Feb", ..., "Nov", "Dec":
df$month <- factor(df$month, levels = unique(df$month))

# Specify "Dec" to be the base level/category:
df$month <- relevel(factor(df$month), ref = "Dec")

# Estimate the model:
m <- lm(avg_bill ~ t + month, data = df)
summary(m)

Exercise 4

We extract the residuals from the model and create a scatter plot of them over time (using the trend term):

library(ggplot2)
df$e <- m$residuals
ggplot(df, aes(t, e)) + geom_point()

We can see clear patterns. For example, between periods 80-100 all the residuals are negative. If the residuals are independent, then this occurring should almost never happen. Therefore there is clear evidence that the residuals display serial correlation.

Exercise 5

To formally test for first-order serial correlation, we estimate a model with the model’s residuals e_t: e_t = \gamma_0 + \gamma_1e_{t-1}+ \nu_t We then apply a t-test (significance test) on \gamma_1. Under H_0 there is no first-order autocorrelation and under H_1 is there is first-order autocorrelation.

We now get the lagged residuals and estimate the auxiliary model in R:

df$lag_e <- c(NA, df$e[1:(nrow(df)-1)])
aux <- lm(e ~ lag_e, data = df)
summary(aux)
