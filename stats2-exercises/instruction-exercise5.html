Instruction Exercise Week 5

Introduction

The file rottentomatoes.csv contains the Rotten Tomatoes rating of a random sample of movies with their length, year of release, and dummy variables for their genres.

  • rating: The movie’s freshness (rating) in percentage terms (between 0 and 100). A higher number is a higher rating.
  • length: The runtime/length of the movie in minutes.
  • year: The year of the movie’s release.
  • comedy: A dummy variable for if the movie is regarded as a comedy or not (=1 if a comedy, =0 if not).
  • action, drama, documentary, romance, horror, thriller. These are defined similarly to comedy.

Note: A movie can have several genres at once. For example, a romantic comedy would have romancer=1 and comedy=1 and would have a zero for all other genres. A movie can also have zero genres, or even 5 genres.

Exercises

Use a 5% significance level for all questions.

  1. Regress rating on length and the square of length.
  • Write down the sample regression equation.
  • Describe the relationship between rating and length using this model.
  • According to this model, at what length is rating maximized on average?
  1. Regress rating on action.
  • Based on this regression, what is the average rating of action movies in the sample?
  • Test using a p-value approach if action movies on average have a more than 3 percentage-point higher rating than non-action movies.
  1. Do action movies on average have a higher rating than non-action movies of the same length? Use a critical value approach.
  2. Regress rating on comedy, length and their interaction. For comedy movies, how much on average does the rating change when the movie is ten minutes longer?
  3. Regress rating on length, year and all the genre dummies. Are the romance and thriller dummies jointly useful? Use a p-value approach.

The sample regression equation is:

\widehat{rating}_i=17.994322 + 0.520754 length_i - 0.001586 length_i^2

Because b_1>0 and b_2<0 and both estimates are individually significant, the relationship between rating and length is mountain-shaped according to this model. The average rating initially increases with length but eventually decreases.

We can find the point at which the rating is maximized on average by differentiating the sample regression equation with respect to length, setting it equal to zero, and finding the length that solves this equation:

\frac{\partial \widehat{rating}_i}{\partial length_i} = 0.520754 - 2\times 0.001586\times length_i =0 Solving: length_i = \frac{0.520754}{2 \times 0.001586}=164.2 We could also calculat this directly in R with:

b_1 <- coef(m1)[2]
b_2 <- coef(m1)[3]
- b_1 / (2 * b_2)

Question 2

The model is: \mathbb{E}[rating_i|action_i]= \beta_0 + \beta_1 action_i For action movies, the average rating is: \mathbb{E}[rating_i|action_i=1]= \beta_0 + \beta_1 For non-action movies, the average rating is: \mathbb{E}[rating_i|action_i=0]= \beta_0 So:

  • \beta_0+\beta_1 is the average rating of action movies
  • \beta_0 is the average rating of non-action movies
  • \beta_1 is the difference in rating between action and non-action movies.

We then estimate the model:

m2 <- lm(rating ~ action, data = df)
summary(m2)

action        4.3794     1.0788    4.06 0.000053 ***

Multiple R-squared:  0.01625,   Adjusted R-squared:  0.01526

To get the average rating of action movies, we need to sum b_0+b_1:

sum(coef(m2))
[1] 57.73864

To confirm this, we can get the average rating of action movies directly as follows:

mean(df$rating[df$action == 1])
[1] 57.73864

We now test if action movies on average have a more than 3 percentage-point higher rating than non-action movies. Because b_1=4.3794 we can see that action movies are estimated to have a 4.3794 percentage-point higher rating. The question is whether this estimate is precise enough to say that it is larger than 3 percentage points in the population (at the 5% level). The claim is thus whether \beta_1>3 in the population. This refers to the alternative hypothesis, so the null hypothesis is the opposite of this (\beta_1\leq 3). We are thus ready to perform our test.

  • The null and alternative hypotheses are: H_0: \beta_1\leq 3, H_1: \beta>3.
  • Under H_0, the test statistic T=\frac{B_1-3}{S_{B_1}}\sim t_{n-k-1}.
  • The value of the test statistic in our sample can be calculated with:
b_1 <- coef(summary(m2))[2, 1]
s_b_1 <- coef(summary(m2))[2, 2]
t <- (b_1 - 3) / s_b_1
t
[1] 1.278705
  • The p-value for this upper-tail test can be calculated with:
pval <- 1 - pt(t, m2$df.residual)
pval
[1] 0.1006492
  • The p-value of 0.1006492 is greater than the significance level, so we fail to reject the null hypothesis. There is insufficient evidence for the claim.

Question 3

To answer this question we need to estimate a model that explains ratin using action and length. To see why, this model is:

\mathbb{E}[rating_i|action_i,length_i]=\beta_0 + \beta_1 action_i + \beta_2 length_i For action movies of a particular length: \mathbb{E}[rating_i|action_i=1,length_i]=\beta_0 + \beta_1 + \beta_2 length_i For non-action movies of the same length: \mathbb{E}[rating_i|action_i=0,length_i]=\beta_0 + \beta_2 length_i Taking differences: \mathbb{E}[rating_i|action_i=1,length_i]-\mathbb{E}[rating_i|action_i=0,length_i]=\beta_1 So \beta_1 in this model measures the difference in average rating between action movies and non-action movies of the same length. If \beta_1>0, then action movies on average have a higher rating than non-action movies of the same length. So the alternative hypothesis is \beta_1>0 and the null hypothesis (the opposite of this) is \beta_1\leq 0. We estimate model:

m3 <- lm(rating ~ action + length, data = df)
summary(m3)

Now we have everything to perform our test.

  • The null and alternative hypotheses are: H_0: \beta_1\leq 0, H_1: \beta>0.
  • Under H_0, the test statistic T=\frac{B_1}{S_{B_1}}\sim t_{n-k-1}.
  • The value of the test statistic in our sample can be seen from the summary output (t=2.970). We could also extract it with:
coef(summary(m3))["action", "t value"]
[1] 2.96966
  • The critical value is:
qt(0.95, m3$df.residual)
[1] 1.646383
  • Because the test statistic is larger than the critical value, we reject the null hypothesis. There is sufficient evidence for the claim.

Question 4

With an interaction, this model is:

\mathbb{E}[rating_i | comedy_i,length_i]= \beta_0 + \beta_1 comedy_i+ \beta_2 length_i + \beta_3 comedy_i length_i For comedy movies, the model becomes: \mathbb{E}[rating_i | comedy_i=1,length_i]= \beta_0 + \beta_1 + (\beta_2 +\beta_3) length_i Increasing the length by 10 minutes: \mathbb{E}[rating_i | comedy_i=1,length_i+10]= \beta_0 + \beta_1 + (\beta_2 +\beta_3) \left(length_i+ 10\right) Taking differences: \mathbb{E}[rating_i | comedy_i=1,length_i+10]- \mathbb{E}[rating_i | comedy_i=1,length_i]= (\beta_2 + \beta_3)\times 10 So to answer this we need to find 10\times(b_2 + b_3) using the estimated b_2 and b_3. The . here means every other variable. Alternatively you could have done: lm(rating ~ length + year + comedy + action + drama + documentary + romance + horror + thriller, data = df) but that involves much more typing.

To check if the romance and thriller dummies are jointly useful we do a partial F test.

  • H_0: \beta_7=\beta_9=0 and H_1: at least one of \beta_7\neq 0 or \beta_9\neq 0.
  • Under H_0, the test statistic: F = \frac{(SSE_r-SSE_c)/(k-g)}{SSE_c/(n-k-1)} follows an F distribution with k-g numerator and n-k-1 denominator degrees of freedom.
  • We can calculate the value of the test statistic as follows. We would normally estimate a reduced model that excludes romance and thriller. We could do this with: lm(rating ~ length + year + comedy + action + drama + documentary + horror, data = df) But this involves a lot of typing. So we will show a trick that also uses the . we used above. If we write rating ~ . - romance - thriller in the formula, it will regress rating on all other variables except romance and thriller. Let’s try it out:
m5_r <- lm(rating ~ . - romance - thriller, data = df)
summary(m5_r)
