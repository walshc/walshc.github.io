R Formula Sheet

Basic R Commands

  • Read in CSV file data.csv as df: df <- read.csv("data.csv")
  • Extract variable x from dataframe df as a vector: df$x

Summary Statistics

  • Mean of x: mean(x)
  • Standard deviation of x: sd(x)
  • Variance of x: var(x)
  • Covariance between vectors x and y: cov(x, y)
  • Correlation between vectors x and y: cor(x, y)
  • Summary statistics of a vector x: summary(x).

Plotting

For all plots, first load the ggplot2 package using library(ggplot2).

  • Scatter plot of variables x and y in dataframe df: ggplot(df, aes(x, y)) + geom_point()
  • Histogram of variable x in dataframe df: ggplot(df, aes(x)) + geom_histogram()

Linear Regression

  • Linear regression of variables y on x1 and x2 contained in dataframe df: lm(y ~ x1 + x2, data = df)
  • Summary output for linear regression model m: summary(m)
  • Extract coefficient estimate on variable x1 from model m: coef(summary(m))["x1", "Estimate"]
  • Extract standard error for variable x1 from model m: coef(summary(m))["x1", "Std. Error"]
  • Obtain 95% confidence interval for coefficient estimates in model m: confint(m, level = 0.95)
  • R^2 from model m: summary(m)$r.squared
  • Adjusted R^2 from model m: summary(m)$adj.r.squared
  • F statistic from model m: summary(m)$fstatistic[1]
  • Extract residuals from model m as a vector: m$residuals
  • ANOVA table for a regression of y on x1 and x2 contained in dataframe df:
m1 <- lm(y ~ 1, data = df)
m2 <- lm(y ~ x1 + x2, data = df)
anova(m1, m2)
  • 95% prediction interval for values x_{p1}=1 and x_{p2}=2 using model m with two independent variables x1 and x2:
df_p <- data.frame(x1 = 1, x2 = 2)
predict(m, df_p, interval = "prediction", level = 0.95)
  • For confidence intervals instead, set interval = "confidence" in the above.
  • Partial F test statistic from a reduced model m_r and complete model m: anova(m_r, r).
  • Linear regression of variables y on x1 contained in dataframe df and its square: lm(y ~ x1 + I(x1^2), data = df)
  • Linear regression of variables y on x1, x2 and their interaction contained in dataframe df: lm(y ~ x1 * x2, data = df)

Critical Values and p-Values

  • Obtain n-k-1 for model m: m$df.residual
  • Obtain quantile p of the t distribution with deg degrees of freedom: qt(p, deg)
  • Obtain the area to the left of q under the t distribution with deg degrees of freedom: pt(q, deg).
  • Obtain quantile p of the F distribution with deg1 numerator and deg2 denominator degrees of freedom: qt(p, deg1, deg2)
  • Obtain the area to the left of q under the F distribution with deg1 numerator and deg2 denominator degrees of freedom: pf(q, deg1, deg2).

Correcting for Heteroskedasticity and Serial Correlation

  • To obtain heteroskedasticity-robust standard errors:
library(lmtest)
library(sandwich)
coeftest(m, vcov = vcovHC(m))
  • To obtain heteroscedasticity and autocorrelation consistent standard errors:
library(lmtest)
library(sandwich)
coeftest(m, vcov = vcovHAC(m))
