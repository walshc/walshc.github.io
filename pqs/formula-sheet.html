Formula Sheet

To obtain help on a function called fn, use the command help(fn) or ?fn in the console.

Arithmetic Operators

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /
  • Exponentiation: ^
  • Modulo (division remainder): %%
  • Integer division (division without remainder): %/%

Mathematical Functions

  • Absolute value: abs(x)
  • Principal square root: sqrt(x)
  • Exponential: exp(x)
  • Logarithm of x to base b: log(x, base = b)
  • Round x to k digits: round(x, digits = k)

Relational Operators

  • Greater than: >
  • Greater than or equal to: >=
  • Less than: <
  • Less than or equal to: <=
  • Equal to: ==
  • Not equal to: !=

Logical Operators

  • Logical constants: TRUE, FALSE
  • Logical negation: !
  • Logical OR: |
  • Logical AND: &

Creating Vectors

  • Combine: c()
  • Sequence of integers from \(a\) to \(b\): a:b
  • Sequence generation: seq()
  • Replicate: rep()
  • Missing value constant: NA

Operations on Vectors

  • Get the 2nd and 4th element from x: x[c(2, 4)]
  • Get x without the 2nd element: x[-2]
  • Get the elements of x where y is TRUE, where y has the same length as x: x[y]
  • Check if missing: is.na()
  • Check if element(s) in a is/are contained somewhere in b: a %in% b
  • Number of elements: length()
  • Summary statistics: summary()
  • Other descriptive functions: sum(), mean(), median(), min(), max()
  • Tabulation and cross-tabulation: table()
  • Convert to character: as.character()
  • Convert to logical: as.logical()
  • Convert to numeric: as.numeric()

Operations on Character Vectors

  • Find and replace: gsub()
  • Remove all non-numeric characters from x: gsub("[^0-9.]+", "", x)
  • Check if pattern is in x: grepl(pattern, x)
  • Convert to lower case: tolower()
  • Convert to factor: factor()
  • Convert x to factor with custom levels mylevels: factor(x, levels = mylevels).

Dates

  • Convert x in format dd/mm/YYYY to date: as.Date(x, format = "%d/%m/%Y")
  • Format options:
    • Day in the month: %d
    • Month as a number: %m
    • Year with century (4 digits): %Y
    • Year without century (2 digits): %y

Operations on Dataframes

  • Get 2nd and 4th rows of df: df[c(2, 4), ]
  • Get rows of df where x is TRUE: df[x, ]
  • Get 2nd and 4th columns of df: df[, c(2, 4)]
  • Get columns of df with names "a" and "b": df[, c("a", "b")]
  • Get column with name a from df: df$a OR df[, "a"] OR df[["a"]]
  • Drop variable called a: df$a <- NULL
  • Drop rows with missing data: df <- na.omit(df)
  • Drop the 2nd variable in df: df <- df[, -2]
  • Convert to dataframe: as.data.frame()
  • Sort by variable x: df <- df[order(df$x, decreasing = FALSE), ]
  • Dimensions: nrow(), ncol(), dim()
  • Name the 2nd column of df to b: names(df)[2] <- "b"
  • Merge df1 with df2 by id variable id: df <- merge(df1, df2, by = "id").
  • Merge df1 with df2 by id variables id1 and id2: df <- merge(df1, df2, by = c("id1", "id2")).
  • Get the mean of x by group g in df: aggregate(x ~ g, FUN = mean, data = df)

Base R Plotting

  • Histogram: hist()
  • Bar plot: barplot()
  • Scatter plot: plot()

Package ggplot2

  • Initialize a ggplot: ggplot(df, aes())
  • Histogram: geom_histogram()
  • Bar plot: geom_bar()
  • Scatter plot: geom_point()
  • Add a fit line to a scatter plot: geom_smooth(method = "lm")
  • Line plot: geom_line()
  • Axis labels: xlab(), ylab()
  • Minimal theme: theme_minimal()

Package reshape2

  • Reshape long to wide: dcast(df, id ~ variable)
  • Reshape wide to long by "id": melt(df, id.vars = "id")

Optimizing and Plotting Functions

  • Creating a function that multiplies its input by 2:
f <- function(x) {
  y <- 2 * x
  return(y)
}
  • Minimize a function f between lower bound lb and upper bound ub: optimize(f, interval = c(lb, ub), maximum = FALSE)
  • Plot a function f() between xmin and xmax:
library(ggplot2)
x <- seq(xmin, xmax, length.out = 2000)
y <- f(x)
df <- data.frame(x, y)
ggplot(df, aes(x, y)) + geom_line()

Conditional Statements

  • Conditional statements for vectors: ifelse(test, value_if_true, value_if_false)
  • Control flow example:

\[ f(x) = \begin{cases} 1.5 & \text{ if } x = 1\\ 3 & \text{ if } x = 2\\ 2x & \text{ otherwise}\\ \end{cases} \]

if (x == 1) {
  return(1.5)
} else if (x == 2) {
  return(3)
} else {
  return(2*x)
}
